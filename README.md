# AIFMapView
## 基于高德地图的几何覆盖图形绘制
1. 支持任意几何图形的描点绘制，支持逐步撤销
2. 支持计算图形的面积，周长
3. 支持替换卫星瓦片底图（需自行下载谷歌瓦片数据）
4. 支持对图形的重叠，或越界判断
5. 支持切换图形或轮廓的填充颜色
